# User Stories

1. As a user, I want to sign up so that I can create an account.
2. As a user, I want to log in so that I can access the application.
3. As a user, I want to see a home screen showing available items.
4. As a user, I want to view details of a selected item.
5. As a user, I want to store my preferences locally.
6. As a user, I want the app to fetch data from an external API.
7. As a user, I want to access a settings menu.
8. As a user, I want to receive notifications from the app.
9. As a user, I want to save favorite items in my profile.
